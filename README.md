# அம்மாவுக்கான பிரீமியம் பிறந்தநாள் இணையதளம்

இந்த project ஒரு முழுமையான தமிழ் Premium Digital Gift website.

## இயக்குவது எப்படி

1. project folder-ஐ VS Code-ல் திறக்கவும்.
2. `index.html`-ஐ browser-ல் திறக்கவும்.
3. அல்லது VS Code Live Server extension பயன்படுத்தலாம்.

## கோப்புகள்

- `index.html` : முழு உள்ளடக்கம் மற்றும் அனைத்து sections
- `style.css` : Premium UI, glassmorphism, responsive design, animations
- `script.js` : interactions, custom cursor, envelope, gallery preview, confetti, fireworks, music engine
- `assets/photos` : demo placeholder images

## உங்கள் புகைப்படங்களை சேர்க்க

- `assets/photos` உள்ள placeholder SVG-க்களை உங்கள் family photos-ஆக மாற்றலாம்.
